# Beauty AI Demo

這是一個可直接放到 GitHub Pages 的靜態網站 Demo。

## 檔案結構

- `index.html`：首頁
- `style.css`：樣式
- `script.js`：AI 彩妝推薦互動效果

## GitHub Pages 使用方式

1. 在 GitHub 新增一個 repository，例如：`beauty-ai-demo`
2. 上傳這個資料夾內的三個檔案到 repository 根目錄
3. 到 Settings → Pages
4. Source 選擇 Deploy from a branch
5. Branch 選擇 main，資料夾選 `/root`
6. 儲存後等待 1–5 分鐘

網址會像這樣：

`https://你的帳號.github.io/beauty-ai-demo/`

如果 repository 名稱是 `你的帳號.github.io`，網址才會是：

`https://你的帳號.github.io/`
